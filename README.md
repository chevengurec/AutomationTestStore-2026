### Тест-кейс TC-01: Проверка сортировки товаров в категориях

**Предусловия**:
Открыта главная страница магазина https://automationteststore.com/. Корзина пуста.

**Тестовые данные**:
Для тестирования используются следующие комбинации: категория "Makeup" и подкатегория "Cheeks" с сортировками "Name A - Z", "Name Z - A", "Price Low > High", "Price High > Low"; категория "Skincare" и подкатегория "Face" с сортировками "Name A - Z" и "Price Low > High"; категория "Books" и подкатегория "Paperback" с сортировкой "Name A - Z".

**Шаги**:

На главной странице навести курсор на категорию, указанную в тестовых данных.

Выбрать подкатегорию, указанную в тестовых данных.

В выпадающем списке сортировки выбрать тип сортировки, указанный в тестовых данных.

Получить список названий товаров (или список цен) после применения сортировки.

Проверить, что полученный список отсортирован в соответствии с выбранным правилом.

**Ожидаемый результат**:
При выборе сортировки "Name A - Z" товары отображаются в алфавитном порядке по названию от А до Z. При выборе "Name Z - A" товары отображаются в обратном алфавитном порядке от Z до А. При выборе "Price Low > High" товары отображаются по возрастанию цены от дешёвых к дорогим. При выборе "Price High > Low" товары отображаются по убыванию цены от дорогих к дешёвым.

### Тест-кейс TC-02: Проверка поиска, добавления товаров в корзину и финальной суммы

**Предусловия**:
Открыта главная страница магазина https://automationteststore.com/. Корзина пуста.

**Тестовые данные**:
Выполняется для поискового запроса: "shirt". Количество добавляемых единиц товара выбирается случайным образом. Для запроса "shirt" выбираются второй и третий товары в выдаче (индексы 1 и 2).

**Шаги**:

В поле поиска ввести тестовый поисковый запрос и нажать клавишу Enter.

Отсортировать результаты поиска по наименованию в алфавитном порядке (Name A - Z).

Кликнуть по названию второго товара в выдаче, чтобы перейти на его страницу.

На открывшейся странице товара установить случайное количество для добавления и нажать кнопку "Add to Cart". После этого происходит автоматический переход на страницу корзины.

Убедиться, что открылась страница корзины.

Вернуться назад к результатам поиска, нажав кнопку "Назад" в браузере дважды.

Кликнуть по названию третьего товара в выдаче, чтобы перейти на его страницу.

На странице товара установить случайное количество для добавления и нажать кнопку "Add to Cart". Происходит переход на страницу корзины.

На странице корзины определить самый дешёвый товар по цене за единицу.

Удвоить количество этого товара и нажать кнопку "Update" для обновления корзины.

Самостоятельно рассчитать ожидаемую итоговую сумму корзины как сумму произведений цены и количества для каждого товара.

Сравнить рассчитанную сумму с суммой, отображаемой на странице корзины.

**Ожидаемый результат**:
Страница результатов поиска содержит товары, соответствующие поисковому запросу, и отсортирована по имени по возрастанию. Товары успешно добавляются в корзину с указанным количеством. Самым дешёвым товаром в корзине считается товар с наименьшей ценой за единицу. После удвоения его количества общая сумма корзины пересчитывается. Итоговая сумма корзины, отображаемая на сайте, совпадает с суммой, рассчитанной вручную по правилу «цена × количество» для всех товаров.

### Тест-кейс TC-03: Проверка корзины с добавлением случайных товаров и удалением чётных позиций

**Предусловия**:
Открыта главная страница магазина https://automationteststore.com/. Корзина пуста.

**Тестовые данные**:
Тест выполняется для добавления 5 случайных товаров. Для каждого выбранного товара количество добавляемых единиц выбирается случайным образом.

**Шаги**:

На главной странице выбрать случайное количество случайных товаров из доступных на странице.

Для каждого выбранного товара сгенерировать случайное количество единиц для добавления (от 1 до 10) и добавить их в корзину многократным нажатием кнопки "Add to Cart".

Перейти на страницу корзины.

В таблице товаров удалить все товары, стоящие на чётных позициях по порядку (второй, четвёртый, шестой и так далее).

Самостоятельно рассчитать ожидаемую итоговую сумму корзины как сумму произведений цены и количества для каждого оставшегося товара.

Сравнить рассчитанную сумму с суммой, отображаемой на странице корзины.

**Ожидаемый результат**:
Все выбранные товары добавляются в корзину с заданным количеством. После удаления товаров на чётных позициях они исчезают из корзины, а товары на нечётных позициях остаются с правильным количеством. Итоговая сумма корзины, отображаемая на сайте, совпадает с суммой, рассчитанной вручную по правилу «цена × количество» для оставшихся товаров.
